Firebase y ajustes

🔧 Conectar Firebase
1

Crear proyecto

Entrá a console.firebase.google.com → "Agregar proyecto" → nombre (ej: iglesia-rotaciones) → desactivar Analytics → Crear.

2

Registrar app Web

Ícono </> → registrá una app web → copiá el bloque firebaseConfig.

3

Crear Firestore

"Firestore Database" → "Crear base de datos" → modo prueba → elegir ubicación.

4

Pegar configuración

API Key
Auth Domain
Project ID
Storage Bucket
Sender ID
App ID
5

Publicar en Netlify

netlify.com → registrate gratis → arrastrá este archivo HTML → URL pública para compartir.
